# LPL 自建选手模拟器

一款以 LPL 职业联赛为背景的文字模拟养成游戏。自建一名选手（选位置），从真实职业选手处继承属性，加入一支真实 LPL 战队补齐该位置（队友 = 该队 2026 真实现役 4 人），随队征战全年四大赛事，最终目标是**夺取世界赛冠军**。

## 玩法

```
创建角色 → 继承属性 → 加入战队(补位置)
   → LPL春季赛 → MSI → LPL夏季赛 → 世界赛(夺冠) 🏆
```

### 核心机制

- **9 维 3 组属性**：对线组（操作/反应/发育）、团战组（配合/意识/开团）、深度组（英雄池/心态/适应性）
- **属性继承**：从随机 5 位真实选手中，每位最多继承 1 项属性，朝源选手方向移动 72%
- **对线突破（被动）**：对线组合计差值 ≥15 时触发单线突破，全属性 ×1.2
- **亮剑（主动）**：世界赛 BO5 关键局可使用，对线组 +15 但玩家与队友团战组 ×0.95
- **成绩门槛晋级**：春季 Top2 进 MSI，年度积分 Top4 进世界赛
- **赛制**：春季 Top4 单败 · 夏季 Top6 胜败组双败 · 世界赛小组 BO1 + 淘汰全程 BO5

## 技术栈

React 19 + TypeScript + Vite + TailwindCSS + Zustand + localStorage + Vitest

| 特性 | 实现 |
|------|------|
| 战力模型 | 9 维属性按位置权重加权 + 团队协同 + 战术契合 |
| 胜率函数 | Logistic/Sigmoid 平滑曲线，高斯随机扰动 |
| 比赛引擎 | 单局 / BO1·3·5 系列赛，事件文字播报 |
| 状态管理 | Zustand 全局状态，FSM 管理游戏阶段流转 |
| 持久化 | localStorage 存档/读档 |
| 可视化 | 纯 SVG 自绘 9 维雷达图（零图表库依赖） |
| 测试 | Vitest 覆盖战力/胜率/突破/亮剑/全年赛季 |

## 本地运行

```bash
npm install
npm run dev      # 开发服务器 http://localhost:5173
npm run build    # 生产构建
npm run test     # 单元测试
npm run preview  # 预览构建产物
```

## 部署

- **Vercel / Netlify**：导入仓库，构建命令 `npm run build`，输出目录 `dist`
- **GitHub Pages**：`npm run build` 后发布 `dist/`

## 项目结构

```
src/
├── types/          # 类型定义（9维属性、选手、战队、比赛结果、存档）
├── constants/      # 位置权重、属性元数据、数值参数
├── data/           # 2026 真实选手/战队数据 + 属性生成器 + 继承池
├── engine/         # 游戏逻辑（战力、单局/系列赛、突破/亮剑、赛季编排）
│   └── __tests__/  # Vitest 单元测试
├── store/          # Zustand 全局状态（FSM + 存档）
├── components/     # 头像/属性条/雷达图/选手卡/战队卡/积分榜
├── pages/          # 菜单/创建/继承/选队/赛季中心/结局
└── utils/          # 随机/钳制/格式化/存储
```

## 数值公式速览

```
单人战力 = 对线均×W_laning + 团战均×W_teamfight + 深度均×W_depth
战队战力 = Σ(单人战力) × 协同(0.85~1.10) × 风格契合
胜率     = 1 / (1 + e^((敌战力-我战力)/60))
单局     = 战力 × 高斯(1, 0.12) 比大小
对线突破 = (我对线和 - 敌对线和) ≥ 15 → 全属性×1.2
亮剑     = 对线组+15, 玩家团战组×0.95, 队友团战组×0.95
```
